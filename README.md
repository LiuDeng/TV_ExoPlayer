# TV_ExoPlayer

基于最新版ExoPlayer修改的TV版本的播放器，扩展了FFMPEG模块，支持更多音视频格式。

非常稳定，支持遥控器快进快退。支持自定义片头广告：视频广告、图片广告（Gif、静态图片均支持）等功能，已经适配手机播放，手机操作的其他功能自己可以修改扩展。

持续完善更新中...
